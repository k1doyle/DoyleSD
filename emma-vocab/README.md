# Emma's Vocab Sprint

A lightweight, no-build study app for Emma's English vocabulary review.

## What It Does

- Flashcards for each vocab word
- Multiple-choice quiz with mixed question types
- Accuracy and streak tracking
- Automatic missed-word review list
- Full word bank with definitions and example sentences

## How To Use It

1. Open `/Users/kevindoyle/Documents/New project/index.html` in any web browser.
2. Start with the flashcards and reveal answers after guessing.
3. Use the quiz to check retention.
4. Click `Practice missed words` to drill the terms that caused trouble.

## Files

- `index.html` contains the app layout.
- `styles.css` contains the visual design and responsive layout.
- `script.js` contains the vocab data and quiz logic.
